This project presents an interactive COVID-19 Dashboard built using Excel and Power BI to analyze death trends across Countries and time.

🔍 Key Insights
Total Deaths: 294K+
COVID-19 Deaths: 21K+
Average Deaths: 151/day
Maximum Deaths: 2K

📊 Features
State-wise filtering using slicers
Time-series analysis (Year, Quarter, Month, Day)
Donut chart for comparative analysis
Bar chart for state-wise deaths
Line chart for trend visualization

🛠️ Tools Used
Microsoft Excel (Data Cleaning)
Power BI (Dashboard & Visualization)
